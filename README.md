An [EPICS](http://www.aps.anl.gov/epics/) module for support of a range of optical spectrometers from Ocean Insight via V3 OceanDirect API.
Credit: John Dobbins - Cornell-Edu

Additional information:
* [Documentation]
  Download Link for OceanDirect SDK 3.1.0  >> [OceanDirect v3.1.0](https://www.oceanoptics.com/software/oceandirect/?srsltid=AfmBOooCHImYYcybeFCc7m4jEy2WCpnfHw6-QMjmMOQYVALTrvQN3YyT)
  For linux, bin file gets downloaded. To run it use:
  ```sudo ./OceanDirect_3_1_3_Linux64.bin```
  It gets installed in ```/usr/local/OceanOptics/OceanDirect```
